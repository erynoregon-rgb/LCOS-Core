# Release Intent

This package is a toy public release that shows engineering discipline without
publishing private advantage.

## Public intent

- Demonstrate clean primitives.
- Demonstrate testing discipline.
- Demonstrate audit/replay reasoning.
- Demonstrate safe hold/escalate behavior.
- Keep examples synthetic and deterministic.

## Non-intent

- Do not disclose non-public implementation details.
- Do not disclose production configuration.
- Do not publish real traces or operational data.
- Do not grant reuse rights beyond the current license.

## Status

Current license posture: `All rights reserved`.

This preserves copyright and patent strategy flexibility while public release
scope is evaluated. This document is not legal advice.
